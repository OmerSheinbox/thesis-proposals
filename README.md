# Thesis Proposals

This repository contains ideas and implementations for my thesis work. 

## 1 & 2. POMDP Integration
This area focuses on integrating Partially Observable Markov Decision Processes (POMDPs) into LLM agent loops, and is further separated into two implemented ideas:
- **PTBench**: A benchmark and prototype (POMDP_V3) for evaluating risk-sensitive tool calling.
- **TASM**: Tool-use As SLAM (Simultaneous Localization and Mapping) - maintaining belief states over world configurations.
- *(Also includes Multi-Agent Dec-POMDP explorations).*

## 3. Bias Handling and Removing
A large, upcoming area of research focused on detecting, handling, and removing bias in AI systems.
