# Thesis Proposals

This repository contains ideas and implementations for my thesis work. 

## 1 & 2. POMDP Integration
This area focuses on integrating Partially Observable Markov Decision Processes (POMDPs) into LLM agent loops, fundamentally divided into two spaces:
- **`01_user_wishes_and_intent`**: Focusing inward on the user. Handling ambiguous prompts, human emotions, urgency, and using risk-sensitive decision making (PTBench, EVPI, Dec-POMDP).
- **`02_tool_usage_and_world_state`**: Focusing outward on the environment. Treating tool usage as noisy sensors to probabilistically map the digital world and handle unreliability (TASM - Tool-use As SLAM).

## 3. Bias Handling and Removing
A large, upcoming area of research focused on detecting, handling, and removing bias in AI systems.
